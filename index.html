SWOT ANALYSIS
// STRENGTH

문제 해결 중심의 상황 대처 역량. 게임 개발 프로젝트 중 기술적 결함으로 일정이 중단되는 위기에서, 사용자 경험을 해치지 않는 대체 로직을 제안해 개발 지연을 단축하고 프로젝트를 완수했습니다.

// WEAKNESS

의견 관철로 인한 협업 유연성 부족. 완성도를 높이려다 팀원과의 충분한 조율이 이루어지지 않아 협업 속도가 일시적으로 지연된 경험이 있습니다.

// OPPORTUNITY

협업과 커뮤니케이션 능력을 중시하는 프로젝트 기반 업무 환경의 증가로, 팀의 시너지를 이끌어내는 역량을 발전시킬 수 있습니다.

// THREAT

의견 관철 성향이 지속될 경우 협업 효율 저하 및 조직 적응에 어려움을 겪을 가능성이 있습니다.

// CONCLUSION

SWOT 분석에서 발견한 나의 '문제 해결 역량'은 2순위 가치인 '업무 몰입도'를 실현하는 핵심 동력이며, 약점인 '의견 관철 성향'을 보완하여 협업의 시너지를 높이는 과정 자체가 1순위 가치인 '성장'을 이루는 밑거름이 될 것이라 확신합니다.

CAREER VALUES
// 1순위 — 성장 가능성

끊임없이 배우고 어제보다 나은 역량을 갖추는 환경. 높은 연봉보다 내 몸값을 높일 수 있는 성장이 장기적으로 더 큰 가치를 창출한다고 믿습니다.

// 2순위 — 업무 몰입도

깊이 있게 파고들어 문제를 해결하고 성취감을 느끼는 과정. 내가 가진 역량을 쏟아부어 문제를 해결하고, 그 과정에서 깊은 몰입을 경험할 때 가장 큰 성취감을 느낍니다.

// 3순위 — 경제적 보상

나의 역량과 성과에 비례하는 정당하고 높은 대가. 성장을 통해 자연스럽게 따라오는 경제적 보상은 나의 전문성을 증명하는 지표가 될 것입니다.
